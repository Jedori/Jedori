exe파일은 윈도우 전용 파일이며 맥 호환은 되지 않습니다.

조작법 (퀘스트 3 컨트롤러 기준)
왼쪽 스틱 - 시점 회전
왼쪽 Y키 - Interactor 끄고 키기 (UI만 상호 작용 가능합니다.)

오른쪽 스틱 - 캐릭터 이동
오른쪽 B키 - 스크린샷

왼쪽 손목을 머리를 향하면 메인 천구 조작 메인 UI가 나옵니다.

왼쪽 부터
										UI 기능
- 천구 조작 UI 토글 버튼 						(현재 자신의 경도 위도, 년, 월, 일, 시간등을 조절하여 조건에 따른 별 위치를 볼 수 있습니다.)
- 별자리 표시기 토글 버튼						(별자리가 어떻게 이어지는지 확인 할 수 있습니다.)
- 인게임 스크린샷 확인 UI 토글 버튼				(스크린샷을 찍고 사진을 저장할 수 있습니다.)
- 별 이동 궤적 그리는 UI 토글 버튼					(시작 시간 부터 끝시간 까지 그려지는 별의 궤적을 볼 수 있습니다.)
이며 Interactor에서 나오는 ray로 눌러도 되고 검지 손가락으로 찌르 듯이 눌러도 됩니다.


절벽을 오르는 구간을 넘어갈 때 정상에 도착하면 앉으면서 몸을 절벽위로 올려 놓듯이 해야 튕겨나가지 않습니다.
퀘스트에서 실행시 버튼은 최대한 살포시 누르거나 interactor로 누르 시는 것을 추천 드립니다.
